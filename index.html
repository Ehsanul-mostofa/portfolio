research

I completed my thesis in the field of IoT and Machine Learning, titled "Enhancing the security of IoT devices using Supervised Machine Learning Algorithms." The goal of my research was to reduce the threats and intrusions that frequently occur on IoT devices, which can lead to issues such as data breaches. We successfully predicted the best possible case to decrease these incoming threats using Naive Bayes, Decision Tree, Random Forest, and Voting Classifier. Additionally, we applied Bagging and Boosting mechanisms such as ADA BOOST and XG BOOST. The implementation was done using Anaconda, a Python distribution and also used popular Python libraries such as Pandas, Scikit-learn, and Numpy.
